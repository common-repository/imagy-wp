== Description ==
サイト高速化サービス imagy（イメージー）のWordPressプラグイン。全自動で画像を圧縮＆キャッシュ＆高速配信を行うサービスです。ご利用にはimagyのお申し込みが必要です。
お申し込みはこちら「https://imagy.jp/」

== Installation ==

1. WP管理パネルから、「プラグイン」 -> 「新規追加」をクリックします。
2. ブラウザの入力ボックスに「imagy WP（イメージー）」と入力します。
3.「imagy WP（イメージー）」プラグインを選択し、「インストール」をクリックします。
4. プラグインを有効にします。
5. WP管理パネルから、「設定」 -> 「imagy高速化設定」をクリックします。
6. 「imagy高をクリックします。
速化設定」画面にて「imagy URL」を入力し「変更を保存」をクリックしてください（imagy URLはimagyの導入手続きが完了致しましたら、弊社よりお送りさせていただきます）。
7. 「imagy 現在の高速化状態」が「有効」となっていることをご確認ください。

もしくは

1.「imagy WP（イメージー）」のダウンロードページから「imagy WP（イメージー）」の.zipファイルをダウンロードします。
3. WP 管理パネルを開き、[プラグイン] -> [新規追加] をクリックします。
4. [プラグインのアップロード] をクリックし、先ほどダウンロードした「imagy WP（イメージー）」の .zip ファイルをアップロードします。
5. [インストール] をクリックし、[プラグインを有効にする] をクリックします。
6. WP管理パネルから、「設定」 -> 「imagy高速化設定」をクリックします。
7. 「imagy高速化設定」画面にて「imagy URL」を入力し「変更を保存」をクリックしてください
    「imagy URL」は、imagyの導入手続きが完了致しましたら弊社よりお送りさせていただきます。
    imagy導入のお申し込みをまだされていない方はこちらのWEBサイトからお申し込みください「https://imagy.jp/」
8. 「imagy 現在の高速化状態」が「有効」となっていることをご確認ください。


== Frequently asked questions ==

=申し込みしてどのくらいで使えるようになりますか？=
申し込み後、最短3営業日で高速化を開始することができます。

=試用期間はありますか？=
はい、最大1ヶ月間の試用期間をご用意しています。
その間でimagyの効果を体感下さい。

=サービスを利用していることを知られたくありません。自社のドメインで運用可能ですか？=
独自ドメインオプション（5,000/初回のみ）をご利用いただくことで自社ドメインの利用が可能です。
その際、DNSの変更権限が必要となります。

=自社で設定するためにはどうしたらいいですか？=
お申込み後、弊社から.htaccessのファイルをお渡しします。
そちらの内容を、御社のファイルに書き加えていただき、動作確認をして下さい。
サポートへご連絡いただければ、動作チェックは行わせていただいております。

=初期設定後は、なにか作業をする必要はありますか？=
設定後は何もしなくても自動的に圧縮＆変換が行われます。

=導入できない場合もありますか？=
一般的なレンタルサーバーでは問題なく動作しますが、サーバーに因っては導入が出来ないケースもございます。自社サーバー等の場合、サポートにお電話いただければ弊社側で調査させていただきます。

=導入が原因で検索順位が落ちることはありませんか？=
Googleは速いサイトを上位に表示するというアルゴリズムを利用しているため、導入することで検索順位が落ちることは有りません。

=導入したことによりブラウザ・デバイスによる表示の不具合はでませんか？=
imagyはアクセスしてくるブラウザを自動的に判別するため、新しいブラウザでも古いブラウザでも問題なく表示が出来ます。
また、古いスマートフォン等も同様に問題なく表示されることを確認しております。
